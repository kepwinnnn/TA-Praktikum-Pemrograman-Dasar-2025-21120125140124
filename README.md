## English 

### What the project does
- Simple **music player** built with PHP.
- Play local music files directly in your browser.

### Why the project is useful
- Organize and play your music collection from your local server.
- Supports album covers if available.

### How users can get started
1. Place the project folder in:
   ```
   /xampp/htdocs/(your-folder-name)/index.php
   ```
   for the music player.
2. Add your music files to:
   ```
   /xampp/htdocs/(your-folder-name)/musics/
   ```
- Album cover images are optional but displayed if available.

### Notes for testing
- You can send multiple songs if needed for testing.
- Most files are `.flac`.
- `.flac` files are large, around 20–30MB per song.
