1. Masukkan file ke folder:  
   ```
   /xampp/htdocs/(namafolder)/index.php
   ```
   untuk music player.
2. Masukkan musik ke folder:  
   ```
   /xampp/htdocs/(namafolder)/musics/
   ```
   - Bisa diatur per album.  
   - Foto cover album bersifat opsional.
